## 格式说明

- __粗体(Bold)__
	- __界面 -> 菜单 -> 子菜单等__ 的标题 
	- 特指 __文件/路径/__ 或 __文件名.txt__.
- _斜体(Italics)_
	- _内容区域_ 需要引起注意
	- __任何粗体内容__ _已经_ __无法再使用斜体__
- `代码块(code)`
	- 单个关键字(词)或组合，比如: `Enter`, `Ctrl+Alt+Del`
	- 命令行语句或者其他打印文字
	- 文件内容的部分引用(包括源代码)，终端输出等等。
